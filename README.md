# Ananth's Portfolio

A modern, responsive portfolio website built with React, Vite, and Framer Motion.
Designed to look premium and high-tech with a focus on user experience.

## Features
- **Responsive Navigation**: Mobile-friendly menu with glassmorphism effect.
- **Hero Section**: Animated introduction with 3D avatar.
- **Projects**: Filterable project gallery with hover effects.
- **Skills**: Visual progress bars and skill categorization.
- **Experience**: Vertical timeline of professional history.
- **Contact**: Functional-looking contact form.

## Tech Stack
- React
- Vite
- Framer Motion (Animations)
- Lucide React (Icons)
- CSS Modules (Vanilla CSS)

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```
